## Estilizar tabela
Estilo de tabela (table)
```
<table class="table">
```

Estilo de tabela com cor / pode ser colocado no table,thead,tr... (table-danger)
```
<table class="table table-danger">
```

Estilo de zebra ou listrado por linha (table-striped)
```
<table class="table table-striped">
```

Estilo de zebra ou listrado por coluna (table-striped-columns)
```
<table class="table table-striped-columns">
```

Quando passo o mouse em cima a cor muda (table-hover)
```
<table class="table table-striped table-hover">
```

Deixar uma linha ativa (table-active)
```
<tr class="table-active">
```

Deixar a tabela com borda em todos os cantos (table-bordered)
```
<table class="table table-striped table-hover table-bordered">
```

Deixar a tabela compacta (table-sm)
```
<table class="table table-striped table-hover table-bordered table-sm">
```

Uma divisoria entre **THEAD** e **TBODY**
```
<tbody class="table-group-divider">
```
## Alinhar Conteudo
Alinhar o conteudo no meio verticalmente (align-middle)
```
<table class="table table-striped table-hover table-bordered table-sm align-middle">
```

Alinhar o conteudo no meio horizontalmente (text-center)
```
<table class="table table-striped table-hover table-bordered table-sm align-middle text-center">
```

## Titulo da tabela
Titulo da tabela
```
<caption>Titulo da tabela</caption>
```

Deixar o titulo no topo (caption-top)
```
<table class="table table-striped table-hover table-bordered table-sm align-middle text-center caption-top">
```

Centralizar titulo (text-center)
```
<caption class="text-center">Titulo da tabela</caption> 
```
