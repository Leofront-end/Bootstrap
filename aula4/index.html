Breakpoints

Os breakpoints no Bootstrap permitem criar layouts responsivos adaptados a diferentes tamanhos de tela. Baseados no conceito "mobile-first", incluem tamanhos como **xs**, **sm**, **md**, **lg**, e **xl**. Esses pontos definem como os elementos se comportam em cada resolução. Por exemplo, a classe `col-md-6` divide a largura em duas colunas a partir de telas médias.
