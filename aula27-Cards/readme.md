# CARDS

Aonde vai ficar o card (card)
```
<div class="card">
    <div class="card-body">
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugiat ipsam ratione quae. Voluptas, placeat. Qui, facere? Consequatur corrupti inventore magni laudantium magnam consectetur excepturi atque vero? Incidunt blanditiis minus saepe!
    </div>
</div>
```

O corpo do card (card-body)
```
<div class="card-body">
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugiat ipsam ratione quae. Voluptas, placeat. Qui, facere? Consequatur corrupti inventore magni laudantium magnam consectetur excepturi atque vero? Incidunt blanditiis minus saepe!
</div>
```

Titulo do card (card-title)
```
<h5 class="card-title">Titulo do card</h5>
```

Subtitulo do card (card-subtitle)
```
<h6 class="card-subtitle">Subtitulo do card</h6>
```

Texto do card (card-text)
```
<p class="card-text">Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugiat ipsam ratione quae. Voluptas, placeat. Qui, facere? Consequatur corrupti inventore magni laudantium magnam consectetur excepturi atque vero? Incidunt blanditiis minus saepe!</p>
```

Links de card (card-link)
```
<a href="#" class="card-link">Link 1</a>
```

Adicionando imagem no topo (card-img-top)
```
<img src="https://picsum.photos/400/400" alt="" class="card-img-top">
```

Adicionando imagem no rodape (card-img-bottom)
```
<img src="https://picsum.photos/400/400" alt="" class="card-img-bottom">
```

Cabeçalho (card-header)
```
<h4 class="card-header">Cabeçalho</h4>
```

Rodapé (card-footer)
```
<div class="card-footer">Rodapé</div>
```

Trocar a cor tematica (text-bg-dark)
```
<div class="card col-4 text-bg-dark">
```
